LLM Research üìö

I'm currently an undergraduate researcher with a focus on Large Language Models (LLMs) and their application in computer science education. Throughout my academic journey, I've been fortunate enough to publish several research papers on the topic, which have garnered recognition from fellow scholars and the wider academic community.

In addition to my own research, I've had the opportunity to lead a talented team of researchers, mentoring them as we collectively explore the cutting-edge realm of LLMs. My passion for this field is fueled by the belief that LLMs hold the potential to completely transform the way we teach and learn computer science, making it more accessible, engaging, and effective for everyone.

As I continue to delve deeper into the world of LLMs, I remain committed to advancing the field by exploring new and innovative ways to harness their power for the benefit of computer science education. I feel incredibly grateful to be part of this exciting journey and can't wait to see what the future holds for LLMs and the learning experiences they can help create.

Generating diverse code explanations using the gpt-3 large language model

This poster explores the potential of LLMs like GPT-3 to support learning by automatically generating diverse natural language explanations for various aspects of a given code snippet, as an alternative to traditional code explanation methods.

Experiences from Using Code Explanations Generated by Large Language Models in a Web Software Development E-Book

This paper explores the integration of LLM-generated code explanations into an interactive e-book on web software development, revealing that students found the explanations helpful, with engagement varying by code complexity, explanation type, and code snippet length.

Automatically Generating CS Learning Materials with Large Language Models

This workshop explores the potential of LLMs in computer science education, addressing their capabilities, implications for academic integrity and curriculum design, and potential integration into pedagogy and research.

The Implications of Large Language Models for CS Teachers and Students

LLMs are increasingly used in classrooms for generating code and learning materials, and this BoF aims to raise awareness about their potential benefits and misuses in computing education while building a globally inclusive learning community.

Big O Optimizer

Big O Optimizer is a Flask-developed web app. It aids developers in optimizing their code according to desired complexities. Integrated with OpenAI's GPT-4, it guarantees precision in automatic code refactoring. The user-friendly interface, enhanced by Bootstrap 4 and Highlight.js, offers immediate feedback for all developers.
